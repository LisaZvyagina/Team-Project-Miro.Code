# Лабораторная работа №4: Base64 кодирование/декодирование

## Цель работы
Изучение и реализация алгоритма кодирования и декодирования base64.

## Требования
- Компилятор C++ (GCC/Clang/MSVC)
- Стандартная библиотека C++

## Сборка и запуск

### Компиляция (Windows)
```bash
g++ main.cpp -o base64_app.exe
```

### Компиляция (Linux/macOS)
```bash
g++ main.cpp -o base64_app
```

### Запуск
```bash
./base64_app.exe  # Windows
./base64_app      # Linux/macOS
```

## Функциональность приложения
1. Кодирование текста/файлов в формат base64
2. Декодирование base64 обратно в исходные данные
3. Обработка стандартного алфавита base64: `ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789+/`
4. Автоматическая вставка символов `=` для выравнивания

## Пример использования
Программа должна поддерживать:
- Чтение данных из файла и кодирование в base64
- Чтение base64 из файла и декодирование
- Прямой ввод/вывод через консоль

## Реализация алгоритма

### Основные этапы кодирования:
1. Чтение по 3 байта исходных данных
2. Разделение на 4 группы по 6 бит
3. Преобразование каждой группы в символ base64-алфавита
4. Добавление символов `=` при необходимости

### Основные этапы декодирования:
1. Чтение по 4 символа base64 (игнорируя символы `=`)
2. Преобразование символов в 6-битные значения
3. Объединение в 3 байта исходных данных

## Контрольные задания
Реализовать кодирование и декодирование для строк:
1. `Xz25` → base64
2. `Fh7k` → base64
3. `gRn4` → base64
4. `Pt54` → раскодировать
5. `Rj0v` → раскодировать
6. `zbTM` → раскодировать
